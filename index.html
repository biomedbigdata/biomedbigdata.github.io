Markus List

Head of the Research Group Big Data in Biomedicine

Chair of Experimental Bioinformatics

Technical University of Munich

Markus List heads the group of Big Data in Biomedicine which is part of the Chair of Experimental Bioinformatics at the Technical University of Munich. We are located in the TUM School of Life Sciences in Freising-Weihenstephan.

We develop predictive models and software tools that integrate big heterogeneous (OMICS) data on the network level. Our long term vision is that our research finds application in clinical practice. Towards this goal, we also address challenges in data privacy as well as in explainability and continuity in modelling.

Interests

  • Integration of BioMedical High-throughput Data
  • Epigenetic mechanisms and cell type heterogeneity
  • Cancer drug target discovery and immunotherapy
  • (Federated) Machine Learning
  • Systems and Network Medicine
  • Computational Models for the Clinics

Projects

CRC1371

The DFG-funded Collaborative Research Center 1371 for Microbiome Signatures investigates the functional relevance of the microbiome in the digestive tract.

FeatureCloud

The EU H2020 project FeatureCloud aims at developing methods for privacy-preserving, federated machine learning.

REPO-TRIAL

The EU H2020 project REPO-TRIAL aims at developing an in silico approach to optimise the efficacy and precision of drug repurposing trials. To this end we integrated heterogeneous data into a comprehensive interactome of disease-drug-gene interactions (a new diseasome).

Sys_CARE

Sys_CARE is a BMBF-funded collaborative research project to elucidate the role of alternative splicing in cardiac and renal diseases using systems medicine approaches and molecular big data.

Software Tools

*
All systems medicine networks microRNAs cell types microbiome HTS
CoVex

A web app for mining the Sars-COV-1⁄2 virus-host-drug interactome for drug repurposing candidates.

BiCoN

A python package and web application for network-constrained biclustering for patient stratification.

WSH

An R package for computing within-sample heterogeneity scores from bisulfte sequencing data.

immunedeconv

An R package for running several tools for in silico cell type deconvolution for immuno-oncology.

HouseKeepR

An R shiny app to mine GEO datasets for tissue- and condition-specific housekeeping genes.

SPONGE

A bioconductor package for inferring genome-wide networks of competing endogenous RNAs.

KPM

A web app for de novo network enrichment, i.e. to integrate multi-omics data with the interactome.

Athena

An R shiny app to mine the Arabidopsis THaliana ExpressioN Atlas across plant tissues.

BioAtlas

A web app to process amplicon data and to study microbial diversity across the globe.

DeepBlueR

A bioconductor package for accessing the DeepBlue epigenome analysis web server.

HiTSeekR

An R shiny web application for the analysis of various types of high-throughput screening data.

JAMI

A fast Java tool for conditional mutual information in ceRNA inference and beyond.

MIRACLE

A web app for sample logistics and data analysis of reverse-phase protein arrays.

SAVANAH

A web app for sample logistics and visual inspection of high-throughput experiments.

OLF

A laboratory information management system for recombinant cell lines in functional genomics.
